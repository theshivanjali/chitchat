# ChitChat

## Social Netowrking Website 

## Languages Used:

* Core PHP
* HTML - CSS
* Bootstrap
* Javascript
* jQuery
* Ajax

## Functionalities:

* Login
* Logout
* Email Validation
* Username Validation
* Encrypted Password System
* Session Management
* URL Routing for Username
* Profile Management
* Post Status
